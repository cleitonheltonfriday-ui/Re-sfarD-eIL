Manual de Operação
⟡ Varredura por A — Fluxo automatizado
1. Abra a página do capítulo no mugiwarasoficial.com
2. Copie a URL da barra de endereços
3. Cole no campo "Varredura por A" e clique em Iniciar
4. O sistema resolve o redirecionamento automaticamente
5. As imagens aparecem em miniatura — clique em Baixar ZIP

O arquivo será nomeado como: 133-one-punch-man.zip
⟢ Link Direto — campanha.php
Se você já tem o link final (campanha.php?auth=...), use esta aba.
O sistema extrai a URL base do CDN e tenta carregar as páginas em ordem numérica: 00, 01, 02, 03...

Caso a numeração comece em 00, ele detecta automaticamente.
⟣ Lote — múltiplos capítulos
Cole quantos links quiser (um por linha). Aceita links de capítulo e links diretos misturados.
O sistema processa cada um sequencialmente, e os ZIPs ficam disponíveis individualmente.

Dica: Você pode adicionar capítulos processados nas abas anteriores ao lote com o botão "+ Adicionar ao Lote".
⚠ Limitações técnicas
Este site roda no seu navegador (GitHub Pages — sem servidor). Para contornar o bloqueio de CORS, usamos proxies públicos.

• Brave: permita pop-ups/redirecionamentos apenas para este site se houver problemas
• Se a varredura automática falhar, use o Link Direto com o URL do campanha.php
• As imagens são baixadas diretamente do CDN do mugiwarasoficial

Nenhum dado é enviado para servidores externos. Tudo ocorre no seu dispositivo.
